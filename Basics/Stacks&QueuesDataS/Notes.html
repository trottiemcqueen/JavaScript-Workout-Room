Stacks & Queues

They are very similar, both are linear data structures also allow us to traverse. That is, go through data elements sequentially one by one. In which only one data element can be directly reached.

The main difference is only how items get removed from this data structure.

Stacks

  1. PUSH(O(1))
  2. PEEK(O(1))
  3. POP(O(1))
  4. LOOKUP(O(n))
